The family tree shown below uses the free, cross platform app called Gramps to manipulate the data. The data can be stored as a .ged file GEDCOM which is supposedly a standard. GEDCOM can be viewed via several different viewers.

Gramps is OK to use once you get used to it, though the included tree viewer isn’t great and it isn’t that intuitive. I always save the file as a .ged after a session, rather than relying on the native .gramps format. Each time you reopen Gramps, the database will still be there but I still make sure to export as a .ged each time I use it.

To get started, download the application and start it up. Under the ‘Family Trees’ menu, select ‘Manage Family Trees’ which opens another window. Create a new empty family tree, then import the .ged file.

Gramps has a pretty wide following, including a Reddit sub-reddit. There are also several add ons available for manipulating the data and better viewing of a family tree.

You can download the .ged file here. Or click here to load the .ged file into an interactive online viewer. This viewer is pretty neat and will reconfigure the tree whenever you click on another person.

NOTE: The file has me as the last descendant and shows trees from both sides of my family. However, you can easily add your own family and prune off unwanted parts of the tree once you become familiar with the application. - Peter
